<h1>My Portfolio</h1>
My personal developer portfolio, a multi-page single-page application showcasing my projects, skills, services, and contact information. Designed and built entirely from scratch with a focus on clean UI, performance, and professionalism.

<h2>Features</h2>
<ul>
<li>Hash-based client-side routing (no React Router)</li>
<li>Fully responsive across all screen sizes</li>
<li>Project showcase with filterable categories</li>
<li>Animated skill bars and timeline</li>
<li>Contact form with validation</li>
<li>Content protection (right-click, copy, DevTools disabled)</li>
<li>Lazy-loaded images for performance</li>
</ul>

**Tech Stack**<br/>
React: Component architecture and state management<br/>
Tailwind CSS: Utility-first styling<br/>
Remixicon: Icon library<br/>
Manrope / Fira Code, Typography<br/>
